TOTEM PARALLAX

Simple mouse and device orientation tracking effect, inspired from Parallax.js

simple mouse tracking with different intensities, use the attribute data-totmov="" on any div you want to move

only x movement, use the attribute data-totmovx="" on any div you want to move

only y movement, use the attribute data-totmovy="" on any div you want to move

different intensities for x movement and y movement, use the attributes data-totmovx="" and data-totmovy="" on any div you want to move
